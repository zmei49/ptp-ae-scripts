# ptp-ae-scripts
IsometricCam v1.0.2 
Что нового в v1.0.2

Секция SCENE ZOOM с живым превью. Слайдер 10000–60000 и три быстрые кнопки Wide / Normal / Tele. Когда двигаешь слайдер — Zoom камеры меняется в реальном времени, видишь результат сразу в окне композиции. Поле ввода тоже работает. Не нужно лезть в свойства камеры в таймлайне. Текущее значение зума учитывается и при первом создании камеры через пресет угла.

Секция LAYER POSE — четыре кнопки для управления выделенным слоем:

    Reset Rot — сбрасывает Orientation, X/Y/Z Rotation в ноль (плюс удаляет все ключи на этих свойствах). Лечит "кривые" повороты, как было у тебя со слоем [ew].
    Lay Flat — кладёт слой на пол изометрической сцены (X Rotation = -90°). Полезно для иконок карт, ковров, наклеек на земле.
    Stand Up — слой стоит вертикально как билборд (все повороты 0).
    Face Cam — самое полезное: Orientation компенсирует углы контроллера, и слой смотрит прямо в камеру, не искажаясь изометрией. Это нужно для текста поверх изометрической сцены, для UI-иконок, для подписей к объектам. Скрипт читает текущие углы X/Y у контроллера и инвертирует их в Orientation слоя.

Защита от ошибок — все pose-операции удаляют ключи перед установкой значения, чтобы не получалось так, что значение не применяется из-за анимации.

Расширенная справка — добавил разделы про Scene Zoom, Layer Pose, и большой раздел "Если картинка стала близко/криво" с пошаговыми инструкциями именно для твоей ситуации.
Как решить твою текущую проблему

    Открой панель v1.0.2
    В секции SCENE ZOOM нажми Normal (вернёт зум на 30000) или подбери слайдером
    Выдели слой [ew] в таймлайне → нажми Reset Rot в Layer Pose — все его кривые повороты обнулятся
    Если хочешь, чтобы [ew] лежал на полу как карта — нажми Lay Flat. Если стоял вертикально — Stand Up. Если это надпись/иконка, которая должна смотреть прямо в зрителя — Face Cam
